FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

Yes — Most of its coding challenges are available without paying, but some premium challenges and extra features require a Pro subscription.

Can I use Frontend Mentor projects in my portfolio?

Yes — you can use Frontend Mentor projects in your portfolio. The official Frontend Mentor FAQ says you're welcome to include anything you build from their challenges.

How can I get help if I'm stuck on a challenge?

If you're stuck on a Frontend Mentor challenge, the official way to get help is by asking in their Discord community where thousands of developers can answer questions.
